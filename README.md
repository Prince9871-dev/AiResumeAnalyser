# CareerLens 🚀

CareerLens is a premium AI-powered Resume Intelligence Platform built to help users optimize their resumes for better job opportunities.

It analyzes resumes using ATS-style evaluation, provides AI-powered personalized feedback, helps match resumes with job descriptions, and improves the chances of getting shortlisted for interviews.

This project was redesigned from a basic resume analyzer into a modern startup-style SaaS product with premium UI/UX, animated backgrounds, glassmorphism, dark-mode design, and a recruiter-impressive portfolio presentation.

---

## ✨ Features

### 📄 Resume Upload & Management

Upload resumes securely and manage multiple submissions in one place.

### 🤖 AI-Powered Resume Analysis

Get intelligent feedback using AI for resume quality, clarity, structure, and improvement suggestions.

### 🎯 ATS Score Evaluation

Analyze how well your resume performs against Applicant Tracking Systems (ATS).

### 🧠 Job Match Intelligence

Compare your resume against target job roles and improve job-specific alignment.

### 📊 Smart Feedback Dashboard

Visual ATS scoring, detailed improvement suggestions, and resume performance tracking.

### 🔐 Secure Authentication

Browser-based authentication powered by Puter.js with no backend setup required.

### ☁️ Cloud Storage

Store uploaded resumes safely using Puter cloud storage.

### 🌌 Premium Startup UI

Modern dark-mode SaaS interface with animated backgrounds, glassmorphism, premium cards, and strong branding.

### 📱 Fully Responsive Design

Optimized experience across desktop, tablet, and mobile devices.

## 🚀 Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/Prince9871-dev/AiResumeAnalyser.git
cd AiResumeAnalyser
```

---

### 2. Install Dependencies

```bash
npm install
```

---

### 3. Run Development Server

```bash
npm run dev
```

---

### 4. Open in Browser

```text
http://localhost:5173
```
